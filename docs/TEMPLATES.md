# Template Overview

Based on research of the best minimal, clean personal websites in 2026.

## Template 1: Founder
**Inspiration:** debmukherjee.com
- **Style:** Professional, light green gradient background
- **Colors:** Soft green (#22c55e), light backgrounds
- **Font:** JetBrains Mono
- **Sections:** Hero, Bio, Experience, Writing, Contact
- **Best for:** Founders, entrepreneurs, startup operators

## Template 2: Developer Dark Mode
**Inspiration:** [fareedmosavat.com](http://fareedmosavat.com/)
- **Style:** Terminal/hacker aesthetic, dark mode
- **Colors:** Deep black (#0a0a0a), neon green (#4af626)
- **Font:** JetBrains Mono (monospace)
- **Sections:** Hero, Career journey, Interactive visualizations
- **Best for:** Engineers, technical founders, developers

## Template 3: Text Scholar
**Inspiration:** [gwern.net](https://gwern.net)
- **Style:** Ultra-minimalist, long-form reading optimized
- **Colors:** Clean white/off-white background, minimal colors
- **Font:** Serif for readability
- **Sections:** Essays, Research, Writing index
- **Best for:** Researchers, writers, academics, deep thinkers

## Template 4: Minimal Clean
**Inspiration:** [paco.me](https://paco.me)
- **Style:** Markdown-style, clean and simple
- **Colors:** Monochrome or near-monochrome
- **Font:** System fonts, highly readable
- **Sections:** About, Work, Projects, Writing
- **Best for:** Designers, minimalists, clean aesthetic lovers

## Template 5: Playful Developer
**Inspiration:** [cassidoo.co](https://cassidoo.co)
- **Style:** Fun with colorful accents, approachable
- **Colors:** Vibrant multi-color palette (green, pink, blue, gold)
- **Font:** Clean sans-serif
- **Sections:** About, Blog, Newsletter, Projects
- **Best for:** Developer advocates, educators, community builders

## Template 6: Blog-First
**Inspiration:** [swyx.io](https://www.swyx.io/)
- **Style:** Content-focused with prominent essays
- **Colors:** Clean, professional palette
- **Font:** Highly readable for long-form
- **Sections:** Latest posts, Popular essays, About, Projects
- **Best for:** Technical writers, bloggers, thought leaders

## Template 7: Resume/CV
**Inspiration:** [One Page Love Resume Collection](https://onepagelove.com/inspiration/resume)
- **Style:** Traditional single-page CV layout
- **Colors:** Professional, often monochrome
- **Font:** Professional sans-serif
- **Sections:** Contact, Skills, Experience, Education, Projects
- **Best for:** Job seekers, consultants, freelancers

## Template 8: Startup
**Inspiration:** YC founder sites + debmukherjee.com variations
- **Style:** Business-focused, achievement-oriented
- **Colors:** Bold, confident colors
- **Font:** Modern sans-serif
- **Sections:** Pitch/Mission, Traction, Team, Contact
- **Best for:** Startup founders, entrepreneurs seeking funding

## Template 9: Creative Portfolio
**Inspiration:** [Minimalist Portfolio Examples](https://reallygooddesigns.com/minimalist-portfolio-website/)
- **Style:** Artistic, photography/design showcase
- **Colors:** Minimal palette to highlight work
- **Font:** Elegant, unobtrusive
- **Sections:** Portfolio grid, About, Contact
- **Best for:** Designers, photographers, artists, creatives

## Template 10: Project Showcase
**Inspiration:** GitHub profiles + developer portfolios
- **Style:** Card/grid layout for projects
- **Colors:** Tech-focused, modern
- **Font:** Code-friendly monospace or clean sans
- **Sections:** Featured projects, Skills, About, Contact
- **Best for:** Product managers, developers with side projects, builders
