# Resume Template

Traditional one-page CV layout optimized for both screen and print.

## Features
- Clean, professional design
- Print-friendly (Ctrl/Cmd + P)
- ATS-compatible
- Mobile responsive

## Best For
- Job seekers
- Freelancers
- Consultants

## Customization
Edit `index.html` to update your information. Designed to fit on one page when printed.
