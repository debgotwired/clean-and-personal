# Blog-First Template

Content-focused design for technical writers and thought leaders.

## Style
- Clean, readable typography
- Essay-first layout
- Minimalist design that highlights content

## Features
- Featured essays section
- Recent posts list
- Newsletter signup
- Dark mode support
- Fully responsive

## Best For
- Technical writers
- Bloggers
- Thought leaders
- Content creators
- Anyone writing regularly

Based on [swyx.io](https://www.swyx.io/)
