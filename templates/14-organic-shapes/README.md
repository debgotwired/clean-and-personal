# Organic Shapes Template

Flowing, natural design with blob shapes and asymmetric layouts breaking from rigid grids.

## Style
- SVG blob shapes
- Flowing curved sections
- Earth-tone color palette
- Asymmetric layouts
- Smooth transitions

## Features
- CSS clip-path for organic containers
- SVG blob backgrounds
- Fluid animations
- Non-grid layouts
- Mobile responsive
