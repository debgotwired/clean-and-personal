# Template 47: Podcast Host

Episode showcase with audio player, guest list, and subscription links for podcast hosts and audio creators.

## Features
- Latest episode with audio player
- Episode grid
- Guest showcase
- Multi-platform subscribe buttons
- Dark theme optimized for podcast branding

## Best For
Podcast Hosts, Audio Creators, Interviewers, Radio Hosts

## Colors
Primary: #8b5cf6 (Purple), Secondary: #ec4899 (Pink), Background: #0f172a (Dark)
