# Glassmorphism Template

Modern portfolio featuring glassmorphism design - a major 2026 trend with frosted glass effects, transparency, and backdrop blur.

## Style
- Frosted glass card effects
- Vibrant gradient background
- Floating animated shapes
- Transparent UI elements
- Backdrop filter blur

## Features
- Glassmorphic UI elements
- Animated floating background shapes
- Gradient background layer
- Sticky glass header
- Glass cards with hover effects
- Contact form with glass inputs
- Mobile responsive
- Scroll animations

## Best For
- Modern portfolios
- Creative professionals
- Designers showcasing trends
- Tech-forward brands
- Anyone wanting contemporary aesthetic

## Customization
Change glass opacity in CSS:
```css
--glass-bg: rgba(255, 255, 255, 0.1);
--glass-border: rgba(255, 255, 255, 0.2);
```

Update background gradient:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 50%, #f093fb 100%);
```
