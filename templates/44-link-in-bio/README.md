# Template 44: Link In Bio

Social hub landing page with stacked links, perfect for Instagram, TikTok, and other social media bio links.

## Features

- **Profile Section**: Avatar, username, and bio
- **Stacked Link Buttons**: Clean, tappable links with icons
- **Social Media Icons**: Quick access to all platforms
- **Mobile-First Design**: Optimized for smartphone screens
- **Gradient Background**: Eye-catching visual design
- **Click Tracking**: Ready for analytics integration
- **Ripple Effects**: Interactive button feedback
- **Fast Loading**: Minimal, optimized code

## Best For

- Influencers
- Content Creators
- Social Media Managers
- Personal Brands
- Musicians & Artists

## Color Scheme

- Primary: `#ec4899` (Pink)
- Secondary: `#8b5cf6` (Purple)
- Accent: `#14b8a6` (Teal)
- Background: `#fafafa` (Light)
- Gradient: Purple to Pink

## Fonts

- Headings: DM Sans (Rounded, friendly)
- Body: Inter (Clean, modern)

## Sections

1. **Profile** - Avatar, name, and bio
2. **Links** - Stacked link buttons
3. **Social** - Social media icons
4. **Footer** - Attribution

## Customization

### Update Profile

```html
<div class="avatar">YI</div>
<h1>@yourname</h1>
<p class="bio">Your bio text here</p>
```

### Add Links

```html
<a href="your-url" class="link-button">
    <span class="link-icon">ðŸ”—</span>
    <div class="link-content">
        <div class="link-title">Link Title</div>
        <div class="link-desc">Link description</div>
    </div>
</a>
```

### Analytics Integration

```javascript
// Google Analytics
gtag('event', 'link_click', {
    link_name: title,
    link_url: url
});

// Bitly or other short link services

```
