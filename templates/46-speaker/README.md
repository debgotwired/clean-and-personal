# Template 46: Speaker

Conference speaker showcase with speaking reel, talk topics, past events, and booking form for keynote speakers and presenters.

## Features
- Speaking reel video section
- Talk topics showcase
- Past events timeline
- Client testimonials
- Media kit download
- Booking inquiry form

## Best For
Conference Speakers, Keynote Speakers, Thought Leaders, Presenters
