# Template 48: Newsletter Writer

Substack-style newsletter writer portfolio with archive, signup form, and subscriber stats for email newsletter publishers.

## Features
- Newsletter signup
- Latest issues showcase
- Full archive
- Subscriber stats
- Author bio
- Reader testimonials

## Best For
Newsletter Writers, Journalists, Essayists, Curators, Analysts

Primary: #f97316 (Orange), Secondary: #1e293b (Slate), Accent: #10b981 (Green)
