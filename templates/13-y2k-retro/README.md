# Y2K Retro Template

Nostalgic 90s/2000s aesthetic featuring chrome effects, neon colors, and early internet vibes.

## Style
- Chrome/metallic text effects
- Neon blue, pink, purple color palette
- Animated star background
- Retro badges and buttons
- Early 2000s web aesthetics

## Features
- Chrome text effect using CSS gradients
- Neon glow effects with text-shadow
- Animated starfield background
- Gradient buttons with hover effects
- Retro visitor counter
- Project cards with overlay
- Timeline with neon styling
- Contact form with neon borders
- Mobile responsive
- Scroll animations

## Best For
- Retro enthusiasts
- Nostalgia-driven portfolios
- Creative professionals
- Designers embracing Y2K trend
- Anyone wanting unique aesthetic

## Customization
Change neon colors in CSS:
```css
--neon-blue: #00d9ff;
--neon-pink: #ff0080;
--neon-purple: #b300ff;
```
