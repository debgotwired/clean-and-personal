# Template 50: Open Source Maintainer

GitHub-focused developer portfolio for open source maintainers with project showcase, contribution stats, and sponsors section.

## Features
- GitHub statistics
- Featured projects with stars/downloads
- Contribution graph
- Sponsor tiers
- Community links
- Recent blog posts
- Dark theme (GitHub-inspired)

## Best For
Open Source Maintainers, Library Authors, Core Contributors, Package Creators

## Colors
Primary: #2da44e (GitHub Green), Secondary: #1f2937, Accent: #0969da (GitHub Blue), Background: #0d1117 (GitHub Dark)

## Fonts
Headings: JetBrains Mono (Developer-focused), Body: Inter

## Sections
1. **Hero** - Name, title, and GitHub profile link
2. **Stats** - Stars, packages, contributors, downloads
3. **Projects** - Featured repositories with metrics
4. **Contributions** - GitHub activity graph
5. **Sponsors** - Sponsorship tiers and current sponsors
6. **Community** - Discord, Twitter, YouTube, Newsletter
7. **Blog** - Recent technical posts
8. **Contact** - Email and social links

## Customization

### Update Stats
Replace the numbers in the stats section with your actual GitHub metrics.

### Add Projects
Duplicate the project card structure and update with your repositories.

### Sponsor Integration
Connect with GitHub Sponsors, Patreon, or Buy Me a Coffee.
